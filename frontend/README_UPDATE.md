# ProfileForge frontend v4 — patch

This archive contains the **contents of the `frontend` folder**.

Apply it by stopping Vite, then replace `src` and `public` in the existing `frontend` folder with the `src` and `public` folders from this archive. Keep `node_modules`.

After replacement run:

```powershell
npm run dev
```

You should see this navigation:

`Главная | Кабинет | Редактор | Проекты | Шаблоны | Публичная`

The header uses `public/assets/profileforge_logo_header.jpg`, created from the supplied team logo.

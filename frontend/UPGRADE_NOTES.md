# Изменения относительно старого статического frontend

Старый `index.html + script.js + styles.css` был визуальным прототипом. Он заменён Vite-приложением на React + TypeScript, так как это соответствует утверждённой архитектуре ProfileForge.

Backend в этом архиве отсутствует и не изменялся. Архив содержит только папку `frontend`.
